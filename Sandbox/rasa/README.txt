Rasa — это платформа машинного обучения с открытым исходным кодом для автоматизации текстовых и голосовых разговоров. С Rasa вы можете создавать контекстных помощников на:

    Facebook Messenger
    Slack
    Google Hangouts
    Webex Teams
    Microsoft Bot Framework
    Rocket.Chat
    Mattermost
    Telegram
    Twilio
    Your own custom conversational channels


или голосовые помощники, такие как:

    Alexa Skills
    Google Home Actions


Rasa помогает вам создавать контекстных помощников, способных вести многоуровневые беседы с большим количеством двусторонних разговоров. Чтобы человек мог иметь содержательный обмен информацией с контекстным помощником, помощник должен иметь возможность использовать контекст для построения вещей, которые обсуждались ранее — Rasa позволяет вам создавать помощников, которые могут делать это масштабируемым образом.

В этом сообщении блога гораздо больше справочной информации.

Обязательно использовать Python 10 версии
Команда для установки библиотеки RASA - pip install rasa
